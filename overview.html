1. System overview

The CDTK is based on the QGIS Server technology and produces web services that follow the OGC standards. Vector datasets are stored in a PostGIS database while raster datasets can be hosted on any source available publicly on the internet.

  • Open-Source solution.

  • Containerized deployment in AWS Cloud.

  • Support for OGC W*S and OGC APIs Standards.
